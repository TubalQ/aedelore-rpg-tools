================================================================================
                          AEDELORE - CLASS ABILITIES
================================================================================

Abilities are special actions available to melee classes (Warrior, Thief/Rogue,
Hunter, Outcast). They cost Weakened points to use and grant bonus dice (Gain).

================================================================================
LEGEND
================================================================================

Weakened    = Cost in Weakened points to use ability (or special condition)
Gain        = Modifier bonus when using ability (e.g., +3 = add 3 to roll)
Check       = Required skill check and DC (1D20 + stat modifier vs DC)
Effect      = What the ability does

USAGE NOTES:
  - "not in battle" = Cannot be used during combat
  - "X rounds" = Effect lasts for X rounds
  - Passive abilities are always active
  - Weakened points are restored after rest

STARTING ABILITIES BY CLASS:
  - Warrior: Last Stand, Hero, Battle Cry
  - Thief/Rogue: Lockpicking, Sneaking, Backstab
  - Hunter: Steady Shot, Tame a Beast, Unveil Path
  - Outcast: Shadow Step, Wilderness Survival, Street Smarts
