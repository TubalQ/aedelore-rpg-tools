Wiki Editor Guide

Content Structure

The wiki is organized in a hierarchy:

  • Books - Top-level categories (e.g., "Lore & History", "Bestiary")
  • Chapters - Sub-sections within a book
  • Pages - Individual articles (can belong to a chapter or be standalone)

Text Formatting & Colors

The wiki automatically applies colors based on HTML tags:

Format TinyMCE Selection Result
Heading 2 Format → Heading 2 Gold text + underline
Heading 3 Format → Heading 3 Cyan text
Blockquote Format → Blockquote Purple left border
Bold Ctrl+B or Bold button Bold text
Italic Ctrl+I or Italic button Italic text

Uploading Images

  1. Open a page for editing
  2. Click the image icon in the toolbar (or drag an image into the editor)
  3. Click Upload tab → Choose file
  4. The image uploads automatically and inserts into your content

Limits: Max 5MB, formats: JPG, PNG, GIF, WebP

Storage: Images are saved to /uploads/images/wiki/YYYY-MM/

Page Fields

Field Description
Title Page name (required). Used for navigation and search.
Summary Shown in search results! Write a brief, descriptive summary (1-2 sentences) that helps users find the page.
Content The main article content. Use the WYSIWYG editor.
Chapter Optional. Assign page to a chapter or leave as standalone.
Sort Order Controls page order within chapter (lower = first).
Tags Comma-separated keywords for filtering (e.g., "creature, undead, boss").
Author Note Attribution or scribe note displayed at top of page.

Tips

  • Use Heading 2 for main sections, Heading 3 for subsections
  • Use Blockquote for in-world quotes, prophecies, or flavor text
  • Tables work great for stats, abilities, and item lists
  • Images are automatically responsive and get rounded corners
  • Changes save when you click "Save Page" - there's no autosave
